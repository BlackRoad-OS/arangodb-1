# Security Policy

## Supported Versions

We release patches for security vulnerabilities for
[supported versions](https://arango.ai/arangodb-product-support-end-of-life-announcements/).

## Reporting a Vulnerability

Please report (suspected) security vulnerabilities to
**[security@arango.ai](mailto:security@arango.ai)**. You will
receive a response from us within 72 hours. If the issue is confirmed,
we will release a patch as soon as possible depending on complexity.
